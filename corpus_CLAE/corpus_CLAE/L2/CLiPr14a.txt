1)	En el aprendizaje de una segunda lengua hay varios métodos de enseñanza que puede usar para adquirirla. Un método, el audio-lingual (ALM) tiene sus raíces en el periodo después de la segunda guerra mundial y ofrece un método que sirvió  a enseñar una lengua rápidamente. Orto método que se revistió en los años setenta fue el método comunicativo lo cual tiene la meta de adquirir una segunda lengua para expresar ideas, actitudes y deseos. Estos dos métodos permiten la enseñanza de una segunda lengua, sin embargo ellos no siempre siguen las mismas reglas y maneras de enseñar.
	El método audio-lingual enfatiza el lenguaje oral y la meta de este método es que los alumnos pueden usar la segunda lengua comunicativamente. En este método, todo el vocabulario está presentado en diálogos, con mucha repetición. Este modelo requiere que los alumnos repite y imita todo los diálogos que el maestro presenta.  Cuando los alumnos hacen errores en su discurso, es el role del maestro para corregirlos inmediatamente.
	Por otro lado, el método comunicativo fue creado después de que los otros métodos fue problemática en algunas maneras. Este método reconoce la necesidad para la competencia lingüística sino también para la competencia comunicativa. Algunas actividades en este método son juegos y diálogos que presentan el uso de la segunda lengua en unos contextos espontáneos. El role del maestro es crear un ambiente que permite mucha conversación y si un alumno comete un error el maestro puede notar el error sin corregirlo.
	Estos dos métodos tienen la misma meta, la adquisición de una segunda lengua pero al mismo tiempo hay diferencias en ambos también.  Primero, el método audio-lingual prohíbe el uso de la lengua nativa, mientras el comunicativo permite el uso de la lengua nativa  pero normalmente usa solamente la segunda lengua para explicar actividades y durante todos los ejercicios. Segundo, en el método ALM es necesario para corregir todos los errores pronto, cuando el estudiante los hace.  Por otro lado el comunicativo declara que es normal para cometer errores y aunque el maestro no los corrige, él nota los errores para discutir en otra lección. Finalmente, el papel del maestro en el método ALM es como un líder o un modelo que los estudiantes pueden imitar; por otro lado, usando el método comunicativo el maestro es más como una persona que guía la lección y se asegura que los estudiantes se entienden la actividad mientras notando errores de los estudiantes. Ambos métodos usan técnicas diferentes para alcanzar el mismo objetivo, teniendo la  habilidad de comunicar en la segunda lengua.

	Competencia comunicativa tiene sus raíces en los años 60 cuando un antropólogo que se daba cuenta de que el código lingüístico no es la única necesidad en la aprendizaje de una segunda lengua. Este concepto fue aplicado en el mundo lingüístico en los años 70 con el uso de este concepto en algunos métodos como el comunicativo.  Se base mucho de este concepto en la habilidades de una persona para diferenciar cuando y cuales son las situaciones apropiadas para usar la lengua. La competencia comunicativa consiste de cuatro componentes incluyendo la competencia gramatical, sociolingüística, discursiva y estratégica. La importancia de este concepto es la competencia de un hablante de una segunda lengua tiene para comunicarse en una manera apropiada en un lugar donde se habla la lengua.
	El primer componente es la competencia gramatical que se refiere al nivel de entendimiento que el alumno tiene con el use del vocabulario, la ortografía, la pronunciación y la formación correcta de palabras y frases. Por ejemplo, la habilidad de un estudiante para usar el vocabulario que aprendió en una frase usando la ortografía correcta, todo sin errores. Por siguiente, el otro componente es la competencia sociolingüístico, este componente incluye las habilidades de usar la gramática y el vocabulario en el contexto adecuado de comunicación. Además, la distinción entre el uso de la lengua en una forma formal como un discurso con su jefe y la menos formal como el lenguaje entre amigos en un el parque, todo esto representa un ejemplo de esta competencia.
	Por añadidura, la competencia discursiva es  la habilidad de formar algunas ideas y expresiones y conectarlas en una forma cohesiva y coherente. En particular, el uso de frases de transición en un dialogo, o argumento, que se dan fluidez a su habilidad discursivo. Para terminar con los componentes, hay la competencia estratégica; el uso de estrategias verbales y no verbales para comunicar su mensaje. La más cómoda o la más competente que se siente con una lengua no necesita usar estas estrategias. Un ejemplo de esta competencia es la perífrasis de una palabra que no sabe el término correcto, o también el uso de gestos para comunicar su mensaje que no puede entender sin estos gestos. Estos componentes forman la competencia comunicativa y la importancia en el aprendizaje de una segunda lengua.
	Este concepto se relacione con el Movimiento para la proficiencia que se basa en la estructura del ACTFL que define niveles de competencia para estudiantes aprendiendo una segunda lengua. Estos dos, la proficiencia y la competencia comunicativa denotan casi la misma cosa. Los niveles en este movimiento, cada uno tiene un nivel de competencia diferente con la segunda lengua que en este caso está decidido por el ACTFL. El nivel más alto en este caso tiene la misma meta, la habilidad para usar la lengua en cualquier situación que se presenta.
